# JuegoRetoUnity

## Descripción
Proyecto desarrollado en Unity 2022.3 LTS correspondiente a la Etapa 1: Fundamentos y Locomoción.

## Funcionalidades
- Movimiento del personaje
- Rotación orientada a cámara
- Salto
- Cámara dinámica con Cinemachine
- Configuración de Layers y Tags

## Controles
- W → avanzar
- S → retroceder
- A → izquierda
- D → derecha
- Mouse → mover cámara
- Barra espaciadora → salto

## Jerarquía de Scripts
Assets/
 └── Scripts/
      └── PlayerMovement.cs

### Descripción
PlayerMovement.cs:
